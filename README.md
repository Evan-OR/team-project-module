# How To Use

## Frontend

1. open terminal and run _cd frontend_
2. run _npm i_
3. then run _npm run dev_
4. enjoy!

## Backend

1. open terminal and run _cd backend_
2. run _npm i_
3. then run _node ._
4. enjoy!
